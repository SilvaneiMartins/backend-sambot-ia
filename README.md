## Execução do projeto

```bash
    # Clone este repositório
    git clone seu_repositório

    # Acessar a pasta do projeto
    $ cd seu_repositório

    # instalar as dependências
    npm i

    # executar o projeto
    npm run dev

    # build o projeto
    npm run build

    # rodar build
    npm run start
```
